CSS - cascading Style Sheets - for the format

  Rule set
    Selector
      Declaration block
        Declaration
          Sets a property to a value

          Rule set example
          header {                   - TAG SELECTOR - THERE CAN BE OTHER TYPES OF SELECTORS
          width:100%;                - {} DECLERATION BLOCK
          height: 100px;             - DECLERATION PROPERTY: VALUE;
          background-color: red;
          }

  Multiple CSS selectors - allowed !
    Can have 1 rule set with mutiple CSS selectors
    * - selects all elements ( wild card)



  Linking CSS to HTML
    1. External stylesheet
        Preferred method
        <link rel="stylesheet" href="main.css">
    2. Internal styles
        <style> </style> - creating a small document and testing it out
    3. Inline styles
        Least preferred = dont worry about it ...
